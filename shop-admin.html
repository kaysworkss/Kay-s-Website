Shop Config

Products

Featured this month

Choose which print is spotlighted in the homepage-style featured section. Visitors will be taken to its detail page if it has a slug.

Discount codes

Create codes customers can apply at checkout. Scope decides what the percentage comes off: Products (item subtotal only), Shipping (delivery only), or Everything (both). A 100%-off code with scope Everything is a free test order it skips payment entirely and records as paid.

Tip: add a code like DEV100 at 100% / Everything to test the full checkout flow (including shipping) without spending anything. Disable it before going live.

Crypto wallet addresses

These are loaded from the shop_config row on the server the same record the server verifies incoming payments against, so the displayed address can never differ from the one being checked. To change them, update eth_address / tezos_address in that row.

Payment keys (set as env vars)

Set PAYSTACK_PUBLIC_KEY in your Vercel environment variables. It is injected at runtime by the page never stored here.

Price auto-conversion

Auto-convert NGN   USD

When on, typing a NGN price fills USD automatically and vice versa. Turn off to set prices independently.
